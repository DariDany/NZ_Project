# NZ Project

This is my first frontend project.

---

## About the project

This is a simple static website built using only:
- HTML
- CSS
- JavaScript (basic usage)

It was created as my first step into frontend development to understand how websites are structured and how basic interactivity works.

---

## Purpose

The goal of this project was to learn:

- How to structure a web page using HTML
- How to style elements using CSS and SASS
- Basic DOM manipulation using JavaScript
- How frontend development works without frameworks

---

## Technologies used

- HTML5
- CSS3
- SASS
- JavaScript (Vanilla JS)

(No frameworks or libraries were used in this project)

---

## What I learned

This was my first experience with frontend development. Through this project I learned:

- How websites are structured
- How styling affects layout and design
- Basic interaction with JavaScript
- How to organize a simple project

---

## Future improvements

If I return to this project, I would improve it by:

- Making it responsive for mobile devices
- Improving UI/UX design
- Refactoring CSS for better structure
- Possibly converting it into a React project
